About me

I am a aspiring Web Developer Student currently (August 2022) at the Bootcamp of Neuefische. This project is our first big recap projects. It is a Quiz Application, where you have Quiz Cards with Questions and Answers. You toggle a button to see the Answers. You can also save favourite Questions with a Bookmark to the Bookmark section of this Web Application. By the tags at the bottom of each Card, you can filter question categories, you want to be shown. Last but not least you have, in th profile section of this Web App a Counter for the amount of Question Cards and also th amount of cards you have bookmarked.

About this project i love the most, the cool design structure i will now use more often per default. I have for exampe, put in the home page area the body element in a grid layout with 3 defined rows for header, main and footer. The main section then i made a flex container as for the card section. I love this because you dont have to worry anymore about padding and margin at the top and the bottom of the page. The scrollable part of the page will only be the main element part. behind the header and the footer (or also Nav as the footer) won't be anything.
